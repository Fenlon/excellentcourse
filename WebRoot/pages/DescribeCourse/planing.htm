特色创新与课程规划

一．本课程的主要特色及创新点
   1. 企业真实项目为载体选取教学内容和组织教学
   2. 以项目为主线而非以知识点为主线，理论知识用工作任务穿起来，理论知识内容在符合工作任务职业行为的同时，也符合学生的认知规律，做到由易到难，由简到繁，飞散难点，前后衔接，循环前进。
  3. 采用了解，熟悉，理解和技能要求等四个层级课程目标表达式。
  4. 与江苏微软技术中心联合授课，把企业最先进的技术和理念传授给学生，实现“零距离”就业。
二、 本课程与国内外同类课程的比较
   当前企业应用开发的重要方向就是基于框架技术的开发应用，本课程培养软件企业需要的高技能人才的职业能力的核心课程，主要培养学生使用流行，成熟的框架技术，开发基于J2EE技术框架的B/S应用系统的职业能力，相对其他同类课程，该门课程掌握的熟练程度是评鉴软件技术专业学生是否合格的标准。
四． 本课程的改建方向与途径
  随着软件开发技术的快速发展，SSH框架表现出安全性，开发过程复杂，功能不够齐全的问题，因此，在下一步课程建设中，对SSH框架进行补充和改进是必要的。具体措施是：
     （1） 在数据存取层（Hibernate）增加了Compass通道存取数据。
     （2） 在MVC（Spring MVC）层，集成了Spring Web Flow开发页面流程，Spring MVC技术，用于Ajax操作组织数据的Spring JSON View，Spring Security 控制安全权限，Junit测试技术等。
     （3） 设计实现了基于DWR的表单验证器，人物动态调度器，引入了事件编程和邮件模板技术，并将这些技术集成到SSH框架中。
三． 本课程的建设目标，步骤及五年内课程资源上网时间表
  近期目标：
  将《J2EE开发技术》课程建设成为校内精品课程的基础上，将其建设成为合格的省内课程课。
   采取措施和步骤为：
  （1） 加强教学队伍建设，加强课程组教学队伍的教学水平，科研水平的提高，加强对精品课程建设的重要性认识的提高，激发课程组教师投身精品课程建设工作的积极性和主动性。
  （2） 对该课程的教学内容和课程体系，实践教学进行进一步的改革，及时反映大学科研领域的最新成果。
